# Twitch Streamer Performance Analysis

## Project Goal
The objective of this project is to analyze the top 1,000 Twitch streamers to identify which factors (specifically **language** and **content maturity**) have the greatest impact on follower growth and viewership retention.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), Matplotlib (Visualization)
* **Platform:** Google Colab / GitHub

## Key Insights
* **Language Trends:** Streamers broadcasting in Spanish, Arabic, Turkish and Portuguese showed a higher average growth rate compared to English-speaking channels.
* **Maturity Rating:** Channels marked as 'Not Mature' had a higher viewer count on an average, suggesting more hype among youngsters.

## How to Run
1. Open the `.ipynb` file in this repository.
2. Click "Open in Colab".
3. Upload the `twitchdata-update.csv` dataset to the runtime.
